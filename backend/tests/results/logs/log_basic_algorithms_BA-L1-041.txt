AI Output:
Sure, here is a simple implementation of the `filter_odd` function in Python:

```python
def filter_odd(arr):
    # Use list comprehension to create a new array with only odd numbers
    return [num for num in arr if num % 2 != 0]

# Test the function
arr = [1, 2, 3, 4, 5]
print("Odd numbers:", filter_odd(arr))

arr = [2, 4, 6, 8, 10]
print("Odd numbers:", filter_odd(arr))
```
