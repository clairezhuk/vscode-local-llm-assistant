Sure, here is a simple implementation of the `has_duplicates` function in Python:

```python
def has_duplicates(arr):
    # Use a set to store unique elements
    unique_elements = set()
    # Iterate over each element in the array
    for item in arr:
        # If the item is already in the set, it means there is a duplicate
        if item in unique_elements:
            return True
        # Otherwise, add the item to the set
        unique_elements.add(item)
    # If no duplicates are found, return False
    return False

# Test the function
arr = [1, 2, 3, 4, 5]
print("Has duplicates:", has_duplicates(arr))
```
