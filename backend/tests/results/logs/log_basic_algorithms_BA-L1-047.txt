Query: Напиши функцію `find_unique(arr)`, яка знаходить єдиний унікальний елемент у масиві, де всі інші повторюються.

Sure, here is a simple implementation of the `find_unique` function in Python:

def find_unique(arr):
    # Use a dictionary to store the frequency of each element
    freq = {}
    # Iterate over the array
    for num in arr:
        # If the element is already in the dictionary, increment its count
        if num in freq:
            freq[num] += 1
        # Otherwise, add it to the dictionary with a count of 1
        else:
            freq[num] = 1
    # Iterate over the dictionary
    for num, count in freq.items():
        # If the count of the current element is 1, return it
        if count == 1:
            return num
    # If no unique element is found, return None
    return None
