Query: Напиши функцію `find_unique(arr)`, яка знаходить єдиний унікальний елемент у масиві, де всі інші повторюються.

Fail Reasons: Execution fail: AssertionError

AI Output:
```python
def find_unique(arr):
    # Use a set to remove duplicates and then find the single remaining element
    unique_element = list(set(arr))[0]
    return unique_element
```
