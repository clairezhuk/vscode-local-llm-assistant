Query: Write a function `linear_search(arr, x)` that retuns the index of x in the array.

AI Output:
Sure, here is a simple implementation of the Linear Search algorithm in Python:

```python
def linear_search(arr, x):
    for i in range(len(arr)):
        if arr[i] == x:
            return i
    return -1

# Test the function
arr = [34, 7, 23, 32, 5, 62, 50]
x = 62
print("Target:", x)
result = linear_search(arr, x)
if result != -1:
    print("Target found at index:", result)
else:
    print("Target not found")
```

In this example, the `linear_search` function takes a sorted list of numbers as input and the target number as input. It uses the Linear Search algorithm to find the target number in the array and returns the index of the target number if it is found, otherwise it returns -1. The Linear Search algorithm works by traversing the array from the first element to the last element and comparing the target value with each element. If the target value is equal to the current element, the function returns the index of the current element. If the target value is not found after traversing the entire array, the function returns -1.
