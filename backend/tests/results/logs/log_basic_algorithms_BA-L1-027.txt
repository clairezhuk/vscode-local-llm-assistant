Query: Write a function `filter_positive(arr)` that returns only the positive numbers from an array.

Fail Reasons: Execution fail: AssertionError

AI Output:
```python
def filter_positive(arr):
    # Use a list comprehension to filter out negative numbers
    positive_numbers = [num for num in arr if num >= 0]
    # Return the filtered list of positive numbers
    return positive_numbers
```
