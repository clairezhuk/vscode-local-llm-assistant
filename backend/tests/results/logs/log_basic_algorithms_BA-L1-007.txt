Sure, here is a simple implementation of the Binary Search algorithm in Python:

```python
def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

# Test the function

if result != -1:
    print("Target found at index:", result)
else:
    print("Target not found")
```

In this example, the `binary_search` function takes a sorted list of numbers as input and the target number as input. It uses the Binary Search algorithm to find the target number in the array and returns the index of the target number if it is found, otherwise it returns -1. The Binary Search algorithm works by repeatedly dividing the search interval in half and comparing the target value with the middle element of the interval. If the target value is equal to the middle element, the function returns the index of the middle element. If the target value is less than the middle element, the function continues the search in the lower half of the interval. If the target value is greater than the middle element, the function continues the search in the upper half of the interval. The search continues until the target value is found or the interval is exhausted.
