Query: Напиши функцію `bubble_sort_desc(arr)`, яка сортує масив за спаданням.

Fail Reasons: Execution fail: AssertionError

AI Output:
```python
def bubble_sort_desc(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] < arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```
