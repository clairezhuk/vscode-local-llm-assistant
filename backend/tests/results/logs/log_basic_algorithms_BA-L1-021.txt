Query: Write a function `fibonacci_array(n)` that returns an array of the first n Fibonacci numbers.

Sure, here is a simple implementation of the `fibonacci_array` function in Python:

```python
def fibonacci_array(n):
    # Initialize the first two Fibonacci numbers
    fib_sequence = [0, 1]
    # Calculate the remaining Fibonacci numbers
    for i in range(2, n):
        fib_sequence.append(fib_sequence[i-1] + fib_sequence[i-2])
    return fib_sequence
```
