Query: Which search algorithm has O(n) time comlexity?

AI Output:
The search algorithm with O(n) time complexity is the Linear Search algorithm. 

Here's a simple explanation of how it works:

1. Start with the first element of the array.
2. Compare the target number with the first element.
3. If the target number is equal to the first element, return the index of the first element.
4. Move to the next element of the array and repeat the process.
5. Continue this process until the target number is found or the entire array is exhausted.

Here is a simple implementation of the Linear Search algorithm in Python:

```python
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# Test the function
arr = [34, 7, 23, 32, 5, 62, 50]
target = 62
print("Target:", target)
result = linear_search(arr, target)
if result != -1:
    print("Target found at index:", result)
else:
    print("Target not found")
```

In this example, the `linear_search` function takes a sorted list of numbers as input and the target number as input. It uses the Linear Search algorithm to find the target number in the array and returns the index of the target number if it is found, otherwise it returns -1.
