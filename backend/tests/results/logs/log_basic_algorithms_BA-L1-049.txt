Query: Write a function `reverse_vowels(s)` that reverses only the vowels in a string.

Fail Reasons: Execution fail: TypeError: 'str' object does not support item assignment

AI Output:
```python
def reverse_vowels(s):
    vowels = "aeiouAEIOU"
    i, j = 0, len(s) - 1
    while i < j:
        while i < j and s[i] not in vowels:
            i += 1
        while i < j and s[j] not in vowels:
            j -= 1
        s[i], s[j] = s[j], s[i]
        i += 1
        j -= 1
    return s
```
