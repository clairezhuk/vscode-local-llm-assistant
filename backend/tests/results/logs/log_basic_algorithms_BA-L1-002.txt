The algorithm that is the fastest way to find a number in a sorted array is the Binary Search algorithm. 

Here's a simple explanation of how it works:

1. Start with the middle element of the array.
2. Compare the target number with the middle element.
3. If the target number is equal to the middle element, return the index of the middle element.
4. If the target number is less than the middle element, repeat the process on the left half of the array.
5. If the target number is greater than the middle element, repeat the process on the right half of the array.
6. Continue this process until the target number is found or the entire array is exhausted.

Here is a simple implementation of the Binary Search algorithm in Python:

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1

# Test the function

target = 6

result = binary_search(arr, target)
if result != -1:
    print("Target found at index:", result)
else:
    print("Target not found")
```

In this example, the `binary_search` function takes a sorted list of numbers as input and the target number as input. It uses the Binary Search algorithm to find the target number in the array and returns the index of the target number if it is found, otherwise it returns -1.
